1. Saat menekan tombol reset, semua input kembali ke nilai awal yang telah ditentukan dalam atribut value. Jika tidak ada value, input akan dikosongkan.
2. jika menekan tombol Kirim Data yang diisi dalam form akan dikirim ke URL yang tertera dalam atribut action (/action_page.php).
3. Atribut target:
- _self : Form terbuka di tab yang sama.
- _blank: Form terbuka di tab baru.
- _parent dan _top: Digunakan jika ada frame dalam halaman web.
4. Atribut method:
- GET: Data dikirim dalam URL (terlihat pada address bar).
- POST: Data dikirim tersembunyi dalam body request.
