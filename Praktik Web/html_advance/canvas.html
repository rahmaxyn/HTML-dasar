KESIMPULAN :
script digunakan untuk menyertakan atau menghubungkan file JavaScript eksternal yang berada di folder js dengan nama main.js. main.js ini akan digunakan untuk mengakses dan memanipulasi elemen-elemen HTML, seperti kanvas, dengan berbagai cara, seperti menggambar grafik atau menangani event yang terjadi pada elemen tersebut. elemen canvas berfungsi sebagai area tempat menggambar atau merender grafik secara dinamis menggunakan JavaScript.Namun, tanpa adanya JavaScript yang terhubung, kanvas ini akan tampak kosong. Dalam konteks ini, file main.js akan menggunakan ID canvas untuk merujuk pada elemen ini dan melakukan operasi menggambar di dalamnya, seperti menggambar bentuk, teks, atau gambar.
