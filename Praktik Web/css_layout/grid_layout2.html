Kesimpulan skenario 1: Dengan mengubah nilai "grid-column-start" dan "grid-column-end", kita dapat mengatur ukuran dan posisi elemen dalam grid dengan lebih fleksibel.
Kesimpulan skenario 2: Dengan mengubah nilai "grid-template-columns", kita dapat mengatur ukuran dan jumlah kolom dalam grid secara fleksibel. Misalnya: - "auto auto" membagi ruang secara merata untuk dua kolom.
- "70px 300px auto 40px" membuat kolom dengan ukuran tetap dan fleksibel.
- "10px 500px 20px 10px" mengatur grid dengan ukuran spesifik.
